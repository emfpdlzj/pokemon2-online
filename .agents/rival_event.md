# 라이벌 등장 이벤트
박사에게 스타팅 몬스터를 받으러 갔을 때 발생하는 "라이벌 등장 이벤트"를 추가해줘.

목표:
- 플레이어가 연구소에서 박사에게 말을 걸기 직전에 라이벌이 등장하는 연출
- 라이벌이 짧은 대사를 하고 연구소 밖으로 나가는 이벤트 구현

구현 요구사항:

1. 이벤트 트리거
- 플레이어가 연구소 내부에서 박사 근처 특정 위치에 도달하면 이벤트 발생
- 이벤트는 1회만 실행되도록 gameState에 플래그 추가 (예: rivalEventTriggered)

---

2. 라이벌 등장 연출
- 이벤트 시작 시 플레이어 입력 잠금
- 라이벌 캐릭터가 화면에 등장 (또는 위치에 생성)
- 플레이어를 바라보게 설정

---

3. 라이벌 대사
다음 대사를 순서대로 출력:

"뭐야 이제야 온거야?"
"늦었네."
"강가에서 기다리고 있을게!"

- 한 줄씩 출력되고 Enter 또는 버튼으로 넘기게 해줘
- 기존 대화 UI 시스템을 그대로 사용

---

4. 라이벌 퇴장
- 대사가 끝나면 라이벌이 연구소 밖 방향으로 이동
- 이동 애니메이션은 간단하게 좌표 이동 또는 위치 변경으로 처리
- 화면에서 사라지면 이벤트 종료

---

5. 상태 변화
- 이벤트 완료 후:
  - gameState.rivalEventTriggered = true
  - 이후 다시 트리거되지 않도록 처리

---

6. 흐름 연결
- 이 이벤트 이후 플레이어는 박사에게 정상적으로 말을 걸 수 있음
- 이후 기존 스타팅 선택 이벤트는 그대로 이어짐

---

7. 코드 구조
- 이벤트 로직은 기존 이벤트 시스템에 맞춰 추가
- 대사 텍스트는 dialogues 데이터 파일로 분리
- 하드코딩 최소화

---

8. 추가 요구
- 라이벌 캐릭터는 기존 NPC 구조를 재사용
- 가능하면 "rival" 타입으로 구분
- 추후 LLM 대화 기능과 연결 가능하도록 구조 열어두기

---

결과:
- 연구소에 들어가면 라이벌이 먼저 등장 → 대사 → 퇴장 → 플레이어가 박사와 상호작용
- 전체 흐름이 자연스럽게 이어지도록 구현
